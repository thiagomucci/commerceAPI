# commerce API

A RESTful commerce API built as part of a personal framework and backend architecture study

## Description
Commerce API provides the core features of a basic e-commerce system, including authentication, product management, and user handling
The project focuses on clean architecture, scalability, and modern tooling

## Tools
Backend: NestJS
Database: PostgreSQL
ORM: Prisma
Frontend: React
Styling: Tailwind CSS

## Freatures

User authentication (Sign up / Sign in)
Product management
Customer (people) management
RESTful API design
Modular and scalable architecture

Getting Started

# install dependencies
npm install

# run database migrations
npx prisma migrate dev

# start development server
npm run start:dev
